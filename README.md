## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/tomasperezdev/push-fe-ht

# Access
$ cd push-fe-ht

# Install dependencies
$ npm install

# Run the project
$ npm start

# The server will initialize in the <http://localhost:3000>
```
